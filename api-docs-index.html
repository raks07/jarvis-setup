Document Management and RAG-based Q&A API Documentation

This page provides links to the API documentation for the Document Management and RAG-based Q&A Application.

jarvis-backend API (NestJS)

The NestJS backend provides user authentication, document management, and ingestion control.

Swagger UI Static Documentation

Key Endpoints

Endpoint Description
/auth/login Authenticate and get JWT token
/auth/register Register new user
/documents Document management endpoints
/users User management endpoints (admin only)
/ingestion Ingestion management endpoints

jarvis-datastore API (FastAPI)

The FastAPI backend provides document processing, embedding generation, and RAG-based Q&A.

Swagger UI ReDoc Static Documentation

Key Endpoints

Endpoint Description
/api/ingestion Document ingestion and processing
/api/qa Question answering endpoints
/api/selection Document selection for Q&A
/health Health check endpoint
